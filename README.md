# OOP school library

> This project consists of Ruby script for the school library app. I implemented encapsulation and inheritance with Ruby by creating classes to represent students and teachers in the school.


## Built With

- Ruby
- Bubocop


## Getting Started

**To get a local copy up and running follow these simple example steps.**

### Prerequisites
- Ruby should be installed on your machine

### Setup
```
git clone https://github.com/zairdon20/OOP-school-library.git
cd OOP-school-library
```
### Install
```
bundle install
```

### Play with the code
```
irb
```

### Run linters
```
rubocop -a
```
